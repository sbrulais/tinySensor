Welcome on Tiny Sensor firmware generation web application

Choose the sensors you implemented on the hardware, provide some identification information for your data, and generate a ready-to-flash optimized firmware.

For now, the only one supported MCU is the Microchip ATTiny84a.

For now, the only one supported protocol is an implementation of the Oregon Scientific protocol.
